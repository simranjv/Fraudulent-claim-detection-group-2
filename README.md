## How to run

1. Open `Report_Fraudulent_Claim_Detection_Anirudha_Bibhudatta_Simran_Vivek_Vivekanand/Fraudulent_Claim_Detection_Anirudha_Bibhudatta_Simran_Vivek_Vivekanand.ipynb` in Jupyter or VS Code.  
2. Install dependencies from the notebook imports (e.g. `pandas`, `numpy`, `matplotlib`, `seaborn`, `sklearn`, `statsmodels`, `imblearn`).  
3. Run all cells from top to bottom (dataset path must match your machine).

## Note

The assignment dataset is referenced inside the notebook; ensure the data file is available at the path used in the first data-loading cells.
